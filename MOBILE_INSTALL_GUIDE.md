
# 📱 How to Install Resume Matcher on Mobile

This guide will help you install the Resume Matcher web app on your mobile phone so it works just like a native app!

## 🚀 Quick Setup

### Step 1: Start the Server on Your Computer
1. Navigate to the project folder: `c:\Users\Sridhar\Desktop\projects\sck`
2. Double-click **`run_app.bat`** to start the server
3. Wait for the message showing:
   ```
   RESUME MATCHER IS STARTING!
   Computer Access: http://127.0.0.1:8080
   Mobile Access:   http://10.226.53.221:8080
   ```
4. **Important:** Note down the "Mobile Access" IP address (e.g., `http://10.226.53.221:8080`)


---

## 📲 For Android Devices

### Step 1: Connect to the Same Wi-Fi
- Make sure your Android phone is connected to the **exact same Wi-Fi network** as your computer

### Step 2: Open the App in Chrome
1. Open **Google Chrome** on your phone
2. Type the **Mobile Access** URL from your computer terminal (e.g., `http://10.226.53.221:8080`)
3. Press Enter to load the website

### Step 3: Install to Home Screen
1. Tap the **3-dot menu** icon (⋮) in the top-right corner of Chrome
2. Select **"Add to Home screen"** or **"Install app"**
3. Give it a name (e.g., "Resume Matcher")
4. Tap **"Add"** or **"Install"**

### Step 4: Launch the App
- Find the **Resume Matcher** icon on your home screen
- Tap it to open the app - it will now run in fullscreen mode like a native app!

---

## 🍎 For iPhone (iOS) Devices

### Step 1: Connect to the Same Wi-Fi
- Make sure your iPhone is connected to the **exact same Wi-Fi network** as your computer

### Step 2: Open the App in Safari
1. Open **Safari** browser (must be Safari, not Chrome)
2. Type the **Mobile Access** URL from your computer terminal (e.g., `http://10.226.53.221:8080`)
3. Tap Go to load the website

### Step 3: Install to Home Screen
1. Tap the **Share** button (the square with an upward arrow) at the bottom of Safari
2. Scroll down and tap **"Add to Home Screen"**
3. Give it a name (e.g., "Resume Matcher")
4. Tap **"Add"** in the top-right corner

### Step 4: Launch the App
- Find the **Resume Matcher** icon on your home screen
- Tap it to open the app - it will now run in fullscreen mode!

---

## ⚠️ Important Notes

### Keep the Server Running
- The app on your phone is a **web app**, so the server on your computer must be running
- Don't close the terminal window on your computer while using the app on your phone

### Accessing from Different Networks
- This setup only works when both devices are on the **same Wi-Fi network**
- If you want to access it from anywhere, you'll need to set up port forwarding (advanced)

### Finding Your IP Address Again
- If you forget the IP address, you can always restart `run_app.bat` and it will show you the Mobile Access URL again

## 🔧 Troubleshooting

### "Cannot connect to server"
- Verify both devices are on the same Wi-Fi
- Check if the server is still running on your computer
- Try accessing the IP address directly in the browser first

### "Add to Home Screen" not showing
- **Android**: Make sure you're using Chrome browser
- **iOS**: Make sure you're using Safari browser (not Chrome)
- Make sure the website has fully loaded before trying to install

### App not opening after installation
- Make sure the server is running on your computer
- Try accessing via browser first, then use the home screen icon
